# IPL First Innings Score Prediction

This project predicts the first innings score range in an IPL match using machine learning.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- VS Code (Jupyter)

## Output
The model predicts a score range based on match conditions such as batting team, bowling team, runs, overs, and wickets.

## How to Run
1. Open the notebook in VS Code or Jupyter
2. Run all cells
3. View the predicted score range in the output
